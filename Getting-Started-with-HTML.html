1. What is WWW (World Wide Web)?

The World Wide Web is a bunch of HTML documents that are connected with hyper links. HTML documents stored in servers can be reached by a user (browser) by HTTP protocols. Intro videos to the web can found here and here . More information about components of the web can be found here .

The following files can be found on the web: plain text, HTML, images, videos and music.

2. What is HTML?

HTML is Hypertext Markup Language. More information can be found here .

3. HTML document structure


Never forget to type at the very beginning of every new HTML document "<!DOCTYPE HTML>" that denotes the document type, as well as opening and closing < html > </html> tags.

< head > < /head > - includes all metadata. < title > < /title > - this what appears in the browser tab. < body > < /body> - is all what is seen in the browser.

4. HTML tags

< name > - opening tag;
< /name > - closing tag;
In between tags there is a content.
The whole structure looks like "< name > content < /name >" and is called element. More information here .

If the closing tag is missing then all the text in the document that follows after the opening tag will be under the influence of the opening tag.

Some tags are only the opening tags and do not require the closing tag, f.x. whitespace tag - < br >.

6. Incline elements and block elements

Some tags, such as < br >, keeps elements in line, another tags, like < p > separates elements into blocks. The blocks have their "private" space with a certain width and length. More information here .

7. Containers elements

Container elements,such as < span > (inline element) and < div > (block element), can contain extra information inside, like text style defined using f.x. CSS.
